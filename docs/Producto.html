Class: Producto

Producto(nombre, imagen, precio, rareza, tipo, bonus)

Clase que representa un producto (objeto) disponible en la tienda. Almacena información sobre armas, armaduras y consumibles.

Constructor

new Producto(nombre, imagen, precio, rareza, tipo, bonus)

Crea una instancia de Producto.
Parameters:
Name Type Description
nombre string Nombre del producto
imagen string URL de la imagen del producto
precio number Precio en monedas (se divide entre 100 para mostrar)
rareza string Nivel de rareza: "comun", "raro" o "epico"
tipo string Tipo de producto: "arma", "armadura" o "consumible"
bonus number Valor de bonificación que aporta el producto
