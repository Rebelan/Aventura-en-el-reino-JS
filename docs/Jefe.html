Class: Jefe

Jefe(nombre, avatar, atk, vida, multiAtkopt)

Clase que representa un Jefe enemigo (enemigo especial con ataque multiplicado). Extiende la clase Enemigo con un multiplicador de ataque.

Constructor

new Jefe(nombre, avatar, atk, vida, multiAtkopt)

Crea una instancia de Jefe.
Parameters:
Name Type Attributes Default Description

atk number Valor base de ataque del jefe
vida number Vida máxima del jefe
multiAtk number <optional>
 1.2 Multiplicador de ataque (por defecto 1.2 = 20% más de daño)
